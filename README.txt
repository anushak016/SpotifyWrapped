HOW TO RUN LOCALLY:
1. Git Clone the repo and open your terminal to the parent folder (Spotify-Wrapped)
2. Once in the terminal, run 'pip install -r requirements.txt'
3. After all the requirements are installed, in the terminal, run 'python manage.py runserver'

1. Visit https://spotify-wrapped-app.onrender.com
